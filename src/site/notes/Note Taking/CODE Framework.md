# The CODE Framework For Note Making
The CODE framework is a method used to capture your thoughts as notes and expand on them designed by Tiago Forte as part of his [Building a Second Brain](https://www.buildingasecondbrain.com/) framwork.

**C**apture - Capture the thought and record it to retrieve later.
**O**rganise - Put it somewhere sensible **AND** link it to other notes so it will be easily accessible when you go looking for it.
**D**istill - What exactly is this thought and why is it important?
**E**xpress - What are your thoughts on this and how does it help others.

This is similar to the [[Note Taking/ENCODE Framework\|ENCODE framework]] and can be used with the [[Note Taking/Idea Compass\|Idea Compass]].
