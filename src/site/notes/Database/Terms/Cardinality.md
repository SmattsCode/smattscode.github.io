# Cardinality
The number of [[Database/Terms/Tuple\|Tuples]] in a [[Database/Terms/Relation\|Relation's]] [[Database/Terms/Body\|Body]], or more simply the number of data rows in a table.

Example:

| UserId | UserName | Email                | Password |
| ------ | -------- | -------------------- | -------- |
| 1      | AJones   | alex.jones@abc.com   | ######## |
| 2      | BSmith   | bob.smith@def.com    | ######## |
| 3      | CBloggs  | chris.bloggs@xyz.com | ######## |

From the table above the Cardinality would be 3.
