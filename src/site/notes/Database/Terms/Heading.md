# Heading
The collection of [[Database/Terms/Attributes\|Attributes]] a [[Databses/Terms/Relation\|Relation]] has.  These are the collection of column headers for a table.

Example:

| ==UserId== | ==UserName== | ==Email==            | ==Password== |
| ---------- | ------------ | -------------------- | ------------ |
| 1          | AJones       | alex.jones@abc.com   | ########     |
| 2          | BSmith       | bob.smith@def.com    | ########     |
| 3          | CBloggs      | chris.bloggs@xyz.com | ########     |

From the table above the highlighted row containing the column names are the Heading.
