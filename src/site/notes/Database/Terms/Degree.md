# Degree
This is the number of [[Database/Terms/Attributes\|Attributes]] in a [[Database/Terms/Relation\|Relations]] [[Database/Terms/Heading\|Heading]], or more simply the number of columns in a table.

Example:

| UserId | UserName | Email                | Password |
| ------ | -------- | -------------------- | -------- |
| 1      | AJones   | alex.jones@abc.com   | ######## |
| 2      | BSmith   | bob.smith@def.com    | ######## |
| 3      | CBloggs  | chris.bloggs@xyz.com | ######## |

From the table above the Degree would be 4.
