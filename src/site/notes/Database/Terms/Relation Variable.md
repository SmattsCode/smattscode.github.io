# Relation Variable or Relvar
The state of a [[Database/Terms/Relation\|Relation]] at a specific point in time.  More simply the contents of a table at a specific point in time.

Example:

User Table @ 13:05:54 on 2026-04-22

| UserId | UserName | Email                | Password |
| ------ | -------- | -------------------- | -------- |
| 1      | AJones   | alex.jones@abc.com   | ######## |
| 2      | BSmith   | bob.smith@def.com    | ######## |

Or:

User Table @ 11:54:17 on 2026-04-23

| UserId | UserName | Email                | Password |
| ------ | -------- | -------------------- | -------- |
| 1      | AJones   | alex.jones@abc.com   | ######## |
| 2      | BSmith   | bob.smith@def.com    | ######## |
| 3      | CBloggs  | chris.bloggs@xyz.com | ######## |
