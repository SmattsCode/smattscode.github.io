# Body
A set of [[Database/Terms/Tuple\|Tuples]] in a [[Database/Terms/Relation\|Relation]], or more simply the data rows in a table excluding the column headers.

Example:

| UserId     | UserName     | Email                    | Password     |
| ---------- | ------------ | ------------------------ | ------------ |
| ==1==      | ==AJones==   | ==alex.jones@abc.com==   | ==########== |
| ==2==      | ==BSmith==   | ==bob.smith@def.com==    | ==########== |
| ==3==      | ==CBloggs==  | ==chris.bloggs@xyz.com== | ==########== |

From the table above the highlighted rows containing data are the body.
