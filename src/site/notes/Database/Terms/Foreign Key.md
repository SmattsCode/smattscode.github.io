# Foreign Key
A unique identifier made up of one or more [[Database/Terms/Attributes\|Attributes]] that links one [[Database/Terms/Relation\|Relation]] to another, establishing a [[Database/Terms/Relationships\|Relationship]] and therefore enforcing [[Database/Terms/Referential Integrity\|Referential Integrity]].  The value of a Foreign Key can also be null and therefore does not link one [[Database/Terms/Relation\|Relation]] to another.
